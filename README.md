# Frizerski salon - Seminarski rad

Aplikacija za zakazivanje termina u frizerskom salonu.

## Tehnologije
- Next.js
- TypeScript
- Tailwind CSS
- Drizzle ORM
- PostgreSQL

## Pokretanje

1. Instaliraj pakete:
npm install

2. Napravi .env.local fajl sa podacima za bazu

3. Pokreni migracije:
npm run db:migrate

4. Ubaci test podatke:
npm run db:seed

5. Pokreni aplikaciju:
npm run dev

## Stranice
- Početna
- Usluge
- Login/Registracija
- Zakazivanje
- Admin panel
- Profil
